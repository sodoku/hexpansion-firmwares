# Moderately Sized Red Button

This is the driver app for the Moderately Sied Red Button hexpansion.

Inserting the hexpansion wih the button depressed will launch a menu, which allows you to select what you want the button to do.

Currently this can act as any of the six frontboard buttons, emit postive and negative emotes and perform resets/shutdowns. More functionality is simple to add.

The chosen setting gets saved to the badge and reloaded on reinsertion. If no setting is found, the select menu is displayed.
