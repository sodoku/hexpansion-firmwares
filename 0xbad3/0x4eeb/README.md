# Keepdexpansion Driver Software

Software to drive the [Keepdexpansion](https://github.com/emfcamp/Keebdexpansion), an official keyboard hexpansion by [@davedarko](https://github.com/davedarko) and [@kliment](https://github.com/kliment).
 
## Key combinations

| Keys | Description |
| --- | --- |
| FN + ESC | Turn leds off|
| FN + SQUARE | Set all leds to red |
| FN + TRIANGLE | Set all leds to orange |
| FN + CROSS | Set all leds to yellow |
| FN + CIRCLE | Set all leds to green |
| FN + CLOUD | Set all leds to blue |
| FN + DIAMOND | Set all leds to purble |
| FN + SOLDERPARTY | Make leds follow tildagon pattern |

SOLDERPARTY is the key next to CTRL
