# Space Unicorn

Hexpansion EEPROM app for the **EMF26 Space Unicorn** — an ATtiny85 I²C slave
(`0x40`) driving WS2812 NeoPixels. Provides a tile-based control UI (pattern,
palette, colour, speed, LED count, pattern parameter) on the Tildagon.

* VID `0x1969` (Corteil) · PID `0x5355` ("SU")
* EEPROM: ST M24256E (256 Kbit / 32 KB)
