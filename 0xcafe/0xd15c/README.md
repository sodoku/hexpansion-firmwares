# Flopagon

The original storage hexpansion.

[Flopagon on GitHub](https://github.com/hairymnstr/Flopagon)

This ID is specifically for the V1 Flopagon which had a 2kB EEPROM.  All V1 Flopagons were **blue**.

You need to short the write-protect jumper to update the EEPROM.  Use tweezers, a jumper wire or even a bent paperclip.
