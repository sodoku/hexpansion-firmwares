# SDHexpansion

A simple Micro-SD card socket to SPI mode hexpansion.  The default firmware for this hexpansion allows loading apps from a folder called sdcard_apps on the root of the drive.

[SDHexpansion on GitHub](https://github.com/hairymnstr/sdhexpansion)

You need to short the write-protect jumper to update the EEPROM.  Use tweezers, a jumper wire or even a bent paperclip.
